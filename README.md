### Configuration
| Environment variable | Description                                                      |
|----------------------|------------------------------------------------------------------|
| DISCORD_TOKEN        | The Discord bot token                                            |
| INVITE_URL           | The URL to invite the bot to a server (used by `invite` command) |
| INTERVAL_MS          | The interval between role colour rotations                       |
